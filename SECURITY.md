# Security Policy

This is an EXPERIMENTAL, test-network-only project. See [DISCLAIMER.md](./DISCLAIMER.md).

## Reporting a Vulnerability

Please report suspected security issues via [GitHub Issues](../../issues) in this repository.
There is currently no private disclosure channel and no bug bounty program.

## Scope Notes

- This project runs exclusively on public test networks; no real funds are ever at risk.
- Do **not** report loss of testnet funds/tokens as a critical or security issue — testnet
  assets have no value and may be reset or drained at any time during development.
- Reports about protocol-logic integrity, signature/verification bypasses, matcher/verifier
  manipulation, or agent-identity spoofing are the most useful reports, even on testnet.
